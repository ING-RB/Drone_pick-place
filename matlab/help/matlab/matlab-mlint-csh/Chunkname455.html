Explanation

Simulink 3D Animation Viewer used by Aero.VirtualRealityAnimation has been removed. Aero.VirtualRealityAnimation does not open or play animations.


Suggested Action

Use sim3d classes in Unreal Engine 3d Environment to create and view a 3D environment instead. For more information on transitioning to the new 3D environment, see Transition Aero.VirtualRealityAnimation to Unreal Engine 3D Environment.
