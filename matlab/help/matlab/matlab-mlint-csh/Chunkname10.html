Explanation

The drange function has either no arguments or more than three. Neither of these scenarios is valid. This is most likely the result of incorrectly converting parfor loops specified prior to MATLAB Version 7.5 (R2007b) to for-drange loops.


Suggested Action

If you specified these parfor loops prior to MATLAB Version 7.5 (R2007b), make the following changes:

Change parfor i=v to for i=drange(v)
Change parfor i=a:b to for i=drange(a,b)
Change parfor i=a:5:c to for i=drange(a,5,b)

For more information, see Looping Over a Distributed Range (for-drange).
